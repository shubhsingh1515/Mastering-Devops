# 📝 Quiz (10 Questions) — Linux Fundamentals

## Questions
1. What does `pwd` display?
2. What command lists files in the current directory?
3. What does `cd ..` do?
4. What is the root directory called?
5. Which command creates a directory?
6. Is `/home/user/docs` an absolute or relative path?
7. What does `~` represent?
8. What is the kernel responsible for?
9. Name two Linux distributions commonly used in DevOps.
10. Why is the terminal preferred in DevOps?

## Answers
1. Present working directory
2. `ls`
3. Moves to the parent directory
4. `/`
5. `mkdir`
6. Absolute
7. Home directory
8. Managing hardware resources and system operations
9. Ubuntu, Debian (also Amazon Linux, Rocky Linux, etc.)
10. It's fast, scriptable, and ideal for remote administration.
