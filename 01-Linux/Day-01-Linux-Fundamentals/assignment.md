# 👨‍💻 Coding/Scripting Exercise

## 1) Create the structure using terminal commands
Create the following structure using **only terminal commands**:

```text
devops-course/

    linux/

        day1/

            notes/

            labs/

            scripts/
```

## 2) Verify it
Run:

```bash
ls -R devops-course
```

# 🔬 Hands-on Lab

## Lab: Set Up Your Practice Workspace

1. Create a `devops-course` directory in your home folder.
2. Inside it, create the directory structure from the exercise.
3. Navigate between each directory using both absolute and relative paths.
4. Return to your home directory using:
   - `cd ~`
5. Run `pwd` after each navigation to confirm your location.
6. Capture the output of:
   - `pwd`
   - `ls -R`

and store it in a text file for your notes.

## Goal
Become comfortable navigating the Linux filesystem without a GUI.
