# Linux Commands (Today)

## 1) Current directory
- `pwd` : Print the current working directory (absolute path).

## 2) List files
- `ls` : List files/directories in the current directory.

## 3) Change directory
- `cd <path>` : Change to a directory.
- `cd ..` : Move to the parent directory.
- `cd ~` : Go to your home directory.

## 4) Create directory
- `mkdir <dir_name>` : Create a new directory.

## 5) Path types (important concept)
- **Absolute path**: Starts from `/` (root), e.g. `/home/user/docs`
- **Relative path**: Starts from your current directory, e.g. `docs`

## Quick reminders
- `/` is the root directory.
- `~` represents your home directory (commonly `/home/<username>`).
- Linux servers are often GUI-less, so the terminal/SSH is used.
