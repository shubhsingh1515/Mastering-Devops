# 📝 Quiz (10 Questions) — Linux File System

## Questions
1. What is the top-most directory in Linux?
2. Which directory contains configuration files?
3. Which directory stores system logs?
4. Which directory contains user home folders?
5. What is the home directory of the root user?
6. Which command creates a symbolic link?
7. Which directory often contains third-party software?
8. Which virtual filesystem exposes process information?
9. Which directory should **not** be used for permanent storage?
10. What does the `d` at the start of `ls -l` output indicate?

## Answers
1. `/` (root)
2. `/etc`
3. `/var/log`
4. `/home`
5. `/root`
6. `ln -s`
7. `/opt`
8. `/proc`
9. `/tmp`
10. Directory
