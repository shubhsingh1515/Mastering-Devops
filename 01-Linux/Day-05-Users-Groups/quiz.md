# 📝 Quiz (10 Questions) — Linux Users & Groups

## Questions
1. Which user has UID 0?
2. Which command displays your current username?
3. Which command creates a group?
4. Which command adds a user to a secondary group?
5. What does `sudo` stand for conceptually?
6. Which command shows user and group IDs?
7. What is the purpose of a service account?
8. Why should applications avoid running as root?
9. How many primary groups can a user have?
10. What is the benefit of using groups?

## Answers
1. `root`
2. `whoami`
3. `groupadd`
4. `usermod -aG`
5. Run a command with administrative privileges
6. `id`
7. To run applications with limited, dedicated permissions
8. To reduce security risks and limit damage if compromised
9. One
10. Easier permission management for multiple users
