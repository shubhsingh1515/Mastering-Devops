# 📝 Quiz (10 Questions) — Linux File Permissions & Ownership

## Questions
1. Which command changes file permissions?
2. Which command changes ownership?
3. What numeric value represents `rwx`?
4. What numeric value represents `rw-`?
5. What permission is recommended for private SSH keys?
6. Which permission allows a script to be executed?
7. What does the first `d` in `drwxr-xr-x` indicate?
8. Which command shows file permissions?
9. Why is `777` discouraged?
10. What is the purpose of `chgrp`?

## Answers
1. `chmod`
2. `chown`
3. 7
4. 6
5. 600
6. `x` (execute)
7. Directory
8. `ls -l`
9. It gives everyone full access, increasing security risk
10. It changes the group ownership of a file or directory
