# 📝 Quiz (10 Questions) — Shell Commands

## Questions
1. Which command copies files?
2. Which command renames a file?
3. Which command displays the last 10 lines of a file?
4. How do you continuously monitor a log?
5. Which command searches for files?
6. Which command shows free disk space?
7. Which command shows directory size?
8. What does `>>` do?
9. Which command displays previous commands?
10. Why is `less` useful?

## Answers
1. `cp`
2. `mv`
3. `tail`
4. `tail -f`
5. `find`
6. `df -h`
7. `du -sh`
8. Appends output to a file
9. `history`
10. It lets you navigate large files page by page
